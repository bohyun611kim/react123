Sigh-up 회원가입 방법이 어떻게 되나요?

A 회원 가입은 코인이즈 PC용 웹페이지서 가능합니다. 회원 가입시 본인명의 이메일 계정이 필요 합니다.

Sigh-up 회원가입 자격이 따로 있나요?

A 네, 있습니다. 미성년자인 경우 가입이 불가합니다

Sigh-up 법인폰으로 이용 가능한가요?

A 법인폰으로는 이용이 불가 합니다

Infomation modification 개인정보를 변경할 수 있나요?

A 마이페이지에서 로그인 비밀번호 변경이 가능하십니다.

Infomation modification 휴대전화 번호를 변경할 수 있나요?

A 휴대전화 번호변경은 불가합니다.

Infomation modification 가입 후 인증레벨을 올리고 싶은데 어떻게 하나요?

A 웹사이트 마이페이지에서 보안인증 또는 신분증 인증을 하시면 됩니다.

Infomation modification 가입시 이름을 잘못 입력하였습니다.

A 1:1 문의에 남겨 주시면 이름을 변경해 드립니다

Infomation modification OTP 설정시 OTP 인증번호가 틀렸다고 표시됩니다

A 구글 OTP 앱에 표시된 6자리 번호가 계속 갱신이 되니 갱신된 번호를 다시 입력해야 합니다

Infomation modification 휴대폰 번호가 변경이 되었습니다.

A 1:1 문의에 OTP/SMS 초기화 신청을 해주신후 보안인증을 재설정해주시기 바랍니다.

Deposit/Withdrawal 코인 지갑 주소는 어디서 확인하나요?

A 지갑관리 입출금페이지에서 입금탭을 클릭하시면 주소확인이 되십니다.

Deposit/Withdrawal 코인 출금 한도는 어떻게

A 인증레벨 별 출금한도이내에서만 가능합니다.

Deposit/Withdrawal 코인 출금수수료는 얼마인가요?

A 출금수수료는 코인별로 상이하며 각 코인별 출금수수료는 수수료안내 페이지를 참조하시면 됩니다.

Deposit/Withdrawal 코인 입출금 시간이 지연되는 이유가 있나요?

A 코인의 입출금 소요 시간은 코인별로 다를 수 있으며 보통 10분~30분 정도 소요 됩니다. 다만 블록체인 네트워크 상에 일시적으로 과도한 입출금 요청이 발생하는 경우 다소 지연될 수 잇습니다

Deposit/Withdrawal 코인 송금 주소를 잘못 입력한 경우 어떻게 하나요?

A 지갑관리-입출금내역 화면에서 코인 입출금내역 조회 후 해당 건의 트랜잭션ID(TID)를 확인 후 고객센터로 문의 주시면 기술적으로 가능한 범위 내에서 복구 해 드립니다. 다만 담당자의 업무량에 따라 시간이 다소 소요될 수 있습니다

Deposit/Withdrawal 코인 입금시 주소외 추가로 입력해야 하는 정보를 누락하고 전송한 경우는 어떻게 하나요?

A 블록체인 트랜잭션ID 나 해시값을 1:1 문의에 문의구분 코인입출금 선택하고 올려주세요. 다만 이 경우 처리하는데 최장 5일까지 소요 될 수 있습니다. 보내신 사이트에서 전송상세 내역을 캡쳐해서 파일로 올려 주셔야 합니다

Deposit/Withdrawal 코인 입금이 하루 이상 지났는데도 처리되지 않고 있습니다. 어떻게 하나요?

A 1:1 문의에 문의구분을 코인입출금으로 선태하신 후 제목에 "코인입금요청합니다 - 입금 후 24시간 경과" 라고 경과 시간을 표시하시면 확인 후 우선 적용 해 드립니다.TXID 또는 Hash 값을 문의 내용에 기입해 주시고 보내신쪽에서 코인 전송내역을 수량이 보이게끔 캡쳐하셔서 파일로 첨부하여 올려 주세요

Deposit/Withdrawal 보유하고 있는 잔고보다 출금가능 수량이 적게 표시됩니다 왜 그런가요?

A 현재 코인 매도 주문을 걸어 놓으신게 없는지 확인 하십시오 미체결 매도 주문이 없는데 그렇다면 1:1 문의에 올려 주세요

Deposit/Withdrawal 보유하고 있는 잔고보다 출금가능 수량이 적게 표시됩니다 왜 그런가요?

A 코인을 거래소간에 송금할시 블록체인 네트워크 상에서 코인 송금 내역을 식별하고 추적할 수 있도록 하는 유일한 키 값입니다. 해당 값이 없으면 코인 송금 내역을 찾는것이 불가능 하며 송금측 거래소 사이트의 코인 출금내역 화면에서 조회 및 확인 할 수 있습니다.

Trading/Orders 거래수수료는 얼마인가요?

A 거래수수료는 매수/매도 공히 0.1%를 적용하고 있습니다.

Trading/Orders 정정주문이 뭔가요?

A 모바일 앱에서만 가능한 정정주문은 기존 매수 또는 매도 주문 내역 중 주문수량 또는 주문가격을 변경하여 재 주문 하는 경우를 말하며 기존 주문내역을 취소하고 새로운 주문을 신규로 내는 번거로움을 없애고자 제공하는 주문 기능입니다.

Trading/Orders 미체결 주문은 어떻게 처리 되나요?

A 주문 취소 전까지는 미체결 상태로 계속 남아 있게 됩니다

Trading/Orders 웹에서 주문 취소가 되지 않습니다

A 일시적인 화면 갱신 장애로 인한 것일 수 있으니 화면 재조회를 해 보시면 취소결과 확인이 가능합니다. 정상화 될 때까지 불편하시더라도 우선 화면 재조회를 사용 하십시오.

Trading/Orders 주문 체결이 되었는데 체결 수량이나 금액이 실제 주문 내역과 일치하지 않습니다

A 1:1 문의에 문의내용 분류를 코인매수/매도 로 선택하시고 내용 남겨 주십시오. 확인 후 답변 드리겠습니다.

Etc 다른 궁금한 사항은 어디에 물어보면 되나요?

A 로그인 하신 후 고객센터-1:1문의하기 화면에서 1:1문의하기 등록 하시면 담당자 확인 후 답변을 드립니다

Etc 신분증 제출시 신분증과 신분증 들고 있는 사진 둘다 제출해야 하나요?

A 네, 맞습니다. 둘 다 제출해야 합니다
